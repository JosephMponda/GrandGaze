# Sustainability Plan

![MUST logo](static/img/must-logo.png) ![GSL logo](static/img/logos/GSL-Official-Logo.png)

Grand Gaze

## Maintenance

- Keep the codebase modular by service area
- Preserve clear ownership of module templates, views, and tests
- Document changes with concise release notes

## Operations

- Support incremental deployment by module
- Optimize for low-bandwidth and intermittent-connectivity environments
- Use standard web stack components that are easy to maintain locally

## Long-Term Growth

- Expand from clinic prototype to specialist-hospital workflows
- Add interoperability layers for national systems in later phases
- Train local maintainers on deployment, data handling, and administration
